# springboot-with-mongodb
A sample Java application built with Spring Boot and integrated with MongoDB as the database. This project demonstrates how to configure, connect, and perform basic CRUD (Create, Read, Update, Delete) operations using Spring Data MongoDB.

---

### Features
- Built with Spring Boot for rapid application development.
- Integrated with MongoDB for flexible, document-based data storage.
- Implements Spring Data MongoDB repositories for database interaction.
- RESTful API endpoints for CRUD operations.
- Configurable database connection settings via application.properties.
